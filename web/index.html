    Über

    Zusammengestellt von Till Dettmering (Follow)

    Dieses Diagramm zeigt die Abhängigkeiten von veschiedenen Parametern zum Mietpreis. Die Farben markieren jeweils einen Stadtteil. Wichtig: Korrelation heißt nicht zwangsläufig Kausation! Verschiedene Faktoren, die hier nicht sichtbar sind, bestimmen den Mietpreis und damit möglicherweise die Korrelation mit den hier gezeigten Parametern.

    Die Daten kommen von Wikipedia, dem Datenportal der Stadt Frankfurt und dem Mietspiegel Frankfurt.

    Based on code by Peter Cook, developed using D3 & Underscore.
